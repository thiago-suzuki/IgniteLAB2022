## Configurações ⚙️
- Node 16.15.0
- Package Yarn
- Porta para testar no Postman ou Insonia: 3000
<br>

## Execuções
- yarn install - Instalar os pacotes
- yarn start:dev -> Executar serviço/aplicação
- yarn test -> Executar os testes
- yarn prisma studio -> Abre o Banco do Prisma no Navegador
